# RewardsHub

RewardsHub is a Vite and React front end for the rewards centre experience.

## Getting Started

Install dependencies:

```bash
npm install
```

Run the local dev server:

```bash
npm run dev
```

Create a production build:

```bash
npm run build
```

## Environment

Copy `.env.example` to `.env` for local configuration. Runtime environment files are ignored by git.
